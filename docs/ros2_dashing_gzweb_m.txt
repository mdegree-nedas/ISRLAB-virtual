	- docker run -it isrlab/ros2_dashing:turtlebot3
		- (follow ROS2 gzweb + models install when you are inside the container)
		- (When you complete the install, exit the container without removing it)

	- docker login

	- docker commit <ros2_container_id> isrlab/ros2_dashing:gzweb_m

	- docker push isrlab/ros2_dashing:gzweb_m


- gzweb + models install (inside Docker container):
	- source /usr/share/gazebo/setup.sh

	- sudo apt update

	- sudo apt install libjansson-dev libboost-dev imagemagick libtinyxml-dev mercurial cmake build-essential

	- curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

	- source ~/.bashrc

	- nvm install 6

	- cd ~; git clone https://github.com/osrf/gzweb
